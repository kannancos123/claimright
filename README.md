# 🚗 ClaimRight

**Car Insurance Claim Guide for India**

A guidance bot for IRDAI-regulated car insurance claims. Get step-by-step help for filing claims, understanding your rights, and navigating the claims process.

## Features

- **10 Claim Scenarios**: Own Damage, Theft, Third-Party, Natural Calamity, Hit-and-Run, Claim Denial, NCB, Cashless vs Reimbursement, Surveyor Dispute, Total Loss
- **Open-Ended Conversation**: Describes your situation naturally and gets guided help
- **IRDAI Compliance**: All guidance aligns with IRDAI regulations
- **Quick Help**: NCB info, complaint process, third-party basics at a glance

## Tech Stack

- **Backend**: FastAPI (Python)
- **Frontend**: Embedded HTML/CSS/JS chat UI
- **Deployment**: Vercel (serverless)

## Quick Start

```bash
pip install fastapi uvicorn pydantic
python server.py
```

Open `http://localhost:8000` in your browser.
